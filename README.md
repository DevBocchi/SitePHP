## 📌 Sobre o Projeto

Esse repositório nasceu como um **espaço de prática e experimentação** com PHP puro (sem frameworks por enquanto).

Aqui eu testo:
- Estrutura de páginas PHP
- Estilização com CSS puro
- Lógica backend básica
- Organização de pastas e includes


É basicamente meu "laboratório PHP" 😄

## 🌐 Tecnologias Utilizadas

- **PHP** (versão 8+)
- **HTML5**
- **CSS3** (estilização em `index.css`)
- Editor/IDE: PhpStorm (por isso a pasta `.idea/`)

Sem frameworks, só PHP nativo + CSS.
